## Example Code

The repositories below contain example code and configuration for use with the Skedulo Pulse Platform.

### Getting Started

All of these repositories require the Skedulo CLI. The Skedulo CLI is a command line interface tool that enables developers to easily build, test, and deploy custom solutions on Skedulo. 

To get started with the CLI and these examples:

* Learn more about the Skedulo CLI on our [Developer Documentation site](https://docs.skedulo.com/developer-guides/cli/skedulo-cli-introduction/)
* [Download and Install](https://docs.skedulo.com/developer-guides/cli/install-the-skedulo-cli/) the Skedulo CLI
* Check out the Skedulo CLI Example repository [linked below](#repositories)
* Follow the instructions on each repository to use and deploy it
